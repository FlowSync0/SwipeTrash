## One-Liner

SwipeTrash is a calm desktop app that helps you clean Documents and Downloads by
swiping through old files: keep what matters, move clutter to the system trash.

## Short Description

SwipeTrash turns file clean-up into a small daily habit. It scans safe candidates
from Documents and Downloads, shows a large preview card, and lets you swipe
right to keep or left to move the file to the system trash. It is open source,
runs on macOS, Windows, and Linux, and avoids hidden files, app folders,
dependency folders, caches, keys, databases, and other risky paths.

## Long Description

Most file-cleaning tools either feel too aggressive or ask you to review huge
lists. SwipeTrash is intentionally slower and safer: it gives you a few old
files per day, shows each one as a large card, then lets you make one simple
decision.

Swipe right to keep the file. Swipe left to move it to the system trash. Nothing
is permanently deleted by the app, and the scanner avoids obvious application,
development, cache, game, and sensitive file paths.

The goal is not to automate deletion blindly. The goal is to make manual clean-up
less annoying, a few decisions at a time.

### Maker Comment

I built SwipeTrash because my Downloads and Documents folders kept filling up,
but classic cleaner apps felt too aggressive. I wanted something slower and more
intentional: a few files per day, one clear preview, one decision.

SwipeTrash does not permanently delete files. It moves them to the system trash,
and it skips risky folders like app data, dependency folders, caches, hidden
files, keys, databases, and obvious game/app assets.

The first release is open source and available for macOS, Windows, and Linux.
Current builds are unsigned, so the OS may show a security warning on first
launch. Feedback on the safety model and UX would be very useful.

## Safety FAQ

### Does SwipeTrash permanently delete files?

No. SwipeTrash moves files to the system trash/recycle bin.

### Which folders does it scan?

By default, it scans Documents and Downloads.

### Does it scan application or system files?

It is designed to skip hidden files, app folders, game/mod assets, dependency
folders, caches, source/build folders, keys, databases, and sensitive names.

### Are the installers signed?

Not yet. macOS Gatekeeper and Windows SmartScreen may warn on first launch until
the project has signing certificates.
